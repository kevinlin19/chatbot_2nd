# chatbot_2nd
python funny chatbot project which is created by tensorflow2.0 keras 
此專案利用自製intent和DNN建立chatbot

**file description:**
- intent.json: 建立主要對話及類別
- data.pickle: 利用nltk取出intent.json訊息，必存於data.pickle中，利於後續使用
- model.bin: DNN 模型
- main.py: 執行檔

**執行方式:** 
```
python main.py
```
